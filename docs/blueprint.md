# **App Name**: BoardView AI

## Core Features:

- Situation Analyzer: Analyzes user's described life, work or business situation using AI to identify key aspects.
- Persona Recommendation: Suggests top 5 relevant public figures for a virtual advisory board based on the situation analysis.
- Board Selection: Allows users to select 3 individuals from the recommendations to form their personal advisory board.
- Facilitated Advice: Simulates advice from the selected board members (Naval, Pieter, GaryVee) based on their known philosophies and approaches.
- Multi-Perspective Synthesis: Combines individual advice into a synthesized action plan, highlighting areas of agreement and disagreement.
- Advisor Customization: Enable user to interact with one advisor, or ask for quick take or a devil's advocate position.
- Telegram Integration: Seamlessly integrates the AI chatbot functionality within the Telegram messaging platform.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke trust and intelligence.
- Background color: Very light desaturated blue (#F0F2F9).
- Accent color: Violet (#736CED) to highlight key actions and elements.
- Headline font: 'Space Grotesk' sans-serif for headlines and short body text. Body text: 'Inter' for longer sections. These pairings bring computerized modernity and ease of reading to the textual portions.
- Code font: 'Source Code Pro' for code snippets in descriptions
- Simple, professional icons to represent advisor profiles and advice categories.
- Subtle animations for transitions between advisor responses.
